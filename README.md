[プロジェクトにアクセス](https://github.com/unic9339/WakuWaku/tree/master/WakuWaku)

### 概要
 •	地域のグルメを登録及び管理しながらウェブサイトの会員情報も管理します。 
それとグルメのレビューを会員達が共有できます。管理者はグルメ情報を管理して会員達に権限を付与しながら管理ができます。
